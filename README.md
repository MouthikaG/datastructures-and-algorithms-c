# Data Structures and Algorithms in C

This repository contains a collection of Data Structures and Algorithms (DSA)
programs implemented using the C programming language.

Each program focuses on a specific DSA concept and is written in a clear,
menu-driven or modular style wherever applicable. The goal of this repository
is to strengthen fundamental understanding through hands-on practice.
